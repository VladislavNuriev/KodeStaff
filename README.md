# Корпоративный список сотрудников

Приложение разрабатывается в рамках тестового задания в компанию KODE в соответствии с требованиями:  
https://github.com/appKODE/trainee-test-android

## Функциональность

Приложение реализует следующие возможности:
- загрузка списка сотрудников;
- фильтрация по поисковому запросу;
- фильтрация по принадлежности к отделу;
- сортировка (по умолчанию - по имени, дополнительно - по дню рождения);
- просмотр профиля сотрудника (по клику на элемент списка).

## Технический стек

### Архитектура
- Многомодульность;
- Clean Architecture

### Сетевой слой
- Retrofit - для выполнения сетевых запросов;
- OkHttp - в качестве HTTP‑клиента.

### Работа с данными
- Coroutines Flow - для обработки и фильтрации данных;
- Room database - модуль для кэширования данных (не подключен в текущей реализации, предусмотрен для масштабирования при увеличении количества сотрудников).

### UI‑слой
- Паттерн MVI + MVVM;
- Jetpack Compose - для отображения данных;
- Androidx Compose Navigation.

### Внедрение зависимостей
- Hilt.
